# 🌿 Grow Craft - Telegram Mini App

## 📱 Установка

### Telegram
1. Откройте бота @BotFather в Telegram
2. Создайте нового бота: `/newbot`
3. Установите Web App: `/setmenubutton` → выберите бота → введите URL: `https://fuckfiat.github.io/grow-craft/`
4. Готово! Кнопка появится в меню бота

### Прямая ссылка
Откройте в Telegram: `https://fuckfiat.github.io/grow-craft/`
