# 🌱 Grow Simulator MEGA

Генетический симулятор выращивания растений с Minecraft-стилем.

## 🎮 Возможности

### 🧬 Генетика
- 5 генов: THC, CBD, урожайность, скорость, устойчивость
- Mendelian наследование при скрещивании
- Доминантные и рецессивные аллели
- Случайные мутации (+5-20%)

### 🔬 Исследования
- 12 технологий в древе
- Автополив, климат-контроль
- CO₂ инъекция, редактирование генов
- Клонирование, мутации

### 🌱 Скрещивание
- Выбери 2 родителя → получи потомство
- Комбинация генов по законам Менделя
- Создавай уникальные штаммы

### 🏪 Магазин
- 100+ товаров
- 12 базовых сортов
- 12 единиц оборудования
- 10 удобрений
- 10 улучшений
- 8 ускорителей
- 8 генетических модификаций

## 🚀 Запуск

### Веб-версия
```bash
npx http-server -p 8080
```
Открой http://localhost:8080

### Electron (десктоп)
```bash
npm install
npm start
```

### Сборка
```bash
npm run build:win    # Windows
npm run build:mac    # macOS
npm run build:linux  # Linux
```

## 📱 PWA

Игра работает как Progressive Web App:
- Устанавливается на устройство
- Работает офлайн
- Автосохранение в localStorage

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| 1-5, 0 | Скорость времени |
| N | Новая игра |
| S | Сохранить |
| F11 | Полный экран |
| Ctrl+/- | Масштаб |
